# 🌌 Galaxy Black Hole Simulation

A highly interactive and visually stunning web-based simulation of a black hole and its surrounding galactic elements. This project renders complex mathematical patterns, gravitational effects, and dynamic particle systems directly in the browser.

## 🚀 Live Demo
Experience the simulation in real-time here: **[Galaxy Black Hole Live](https://pankajtiwari-art.github.io/galaxy-Black-hole/)**

## ✨ Features
* **Dynamic Particle Engine:** Handles moving particles representing stars, cosmic dust, and debris.
* **Simulated Physics:** Uses custom JavaScript logic to simulate orbital mechanics and gravitational pull.
* **High-Performance Rendering:** Optimized to run smoothly in modern web browsers without heavy external libraries.
* **Immersive Visuals:** Features a deep, dark cosmic aesthetic with glowing elements and smooth rotational dynamics.
* **Fully Responsive:** Adapts perfectly to different screen sizes for both desktop and mobile users.

## 🛠️ Technologies Used
* **HTML5:** Canvas and application structure.
* **CSS3:** Deep space background aesthetics and styling.
* **JavaScript (Vanilla):** The core engine handling the physics, math, and frame-by-frame rendering.

## ⚙️ How to Run Locally
1. Clone this repository or download the ZIP file.
2. Extract the folder on your computer.
3. Open the `index.html` file in any modern web browser to view the simulation.
